模块: type

获取对象的类型（boolean，number，string，function，array，date，regexp，object，error，NaN，undefined，null）

Example

// 类型为："string"
$api.type('abc')

新增类型

name string

类型名称
object Object

需要注册类型的对象
Example
class People {
     constructor() {
     // Register the object type.
     $api.type.add('People', this);
